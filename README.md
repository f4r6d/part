# Part Soft Project

## installing:

- git clone https://github.com/f4r6d/part.git
- cd part
- pip install -r requirement.txt
- cd partsoft
- mv .env.sample .env
- open .env file and fill database host and user and pass .. (postgres)
- cd ..
- python3 manage.py migrate
- python3 manage.py runserver
